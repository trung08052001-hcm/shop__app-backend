# Shop App — Backend API

REST API cho ứng dụng e-commerce xây dựng bằng Node.js.

## Tech Stack
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- RESTful API

## API Endpoints
- POST /api/auth/register
- POST /api/auth/login
- GET  /api/products
- GET  /api/products/:id
- GET  /api/products/categories
- POST /api/orders
- GET  /api/orders

## Setup
1. Clone repo
2. npm install
3. Tạo file .env (xem .env.example)
4. npm run dev
